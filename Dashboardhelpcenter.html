❓ FAQ Umum
Apa itu Dashboard Admin TK Aisyiyah? ▾

Dashboard Admin adalah sistem manajemen berbasis web khusus untuk pengelola TK Aisyiyah Timika. Melalui dashboard ini, admin dapat mengelola data profil sekolah, staff pengajar, fasilitas, galeri foto, kalender event, dan pendaftaran PPDB secara terpusat.

Siapa yang bisa mengakses dashboard ini? ▾

Hanya akun dengan hak akses Admin (Super Admin) yang dapat masuk ke dashboard ini. Akses diberikan oleh kepala sekolah atau pengelola utama TK Aisyiyah Timika.

Bagaimana cara login ke dashboard? ▾

Buka halaman Login Admin, masukkan username dan password yang telah diberikan oleh kepala sekolah. Klik tombol "Login". Jika lupa password, gunakan fitur "Lupa Password" atau hubungi administrator sistem.

Bagaimana cara menggunakan fitur pencarian? ▾

Kolom pencarian berada di bagian atas (topbar) di setiap halaman. Ketik kata kunci seperti "Galeri", "PPDB", atau "Fasilitas" untuk langsung diarahkan ke menu yang relevan.

Apakah dashboard ini bisa diakses dari HP? ▾

Ya! Dashboard ini telah didesain responsif dan dapat diakses melalui perangkat mobile (HP/tablet). Gunakan browser modern seperti Chrome, Safari, atau Firefox untuk pengalaman terbaik. Pada perangkat mobile, sidebar dapat dibuka menggunakan tombol menu (☰) di pojok kiri atas.

Bagaimana cara mengirim saran dan masukan? ▾

Klik tombol "💬 Saran dan Masukan" di pojok kanan atas topbar. Sebuah form akan muncul — tuliskan saran atau masukan Anda, lalu klik "Kirim". Masukan Anda sangat berharga untuk pengembangan sistem ini.

📋 Pertanyaan Seputar PPDB
Bagaimana cara membuka PPDB? ▾

Buka menu PPDB 2027/2028 dari sidebar. Klik tombol "Buka PPDB 2027/2028" berwarna hijau. Konfirmasi tindakan pada dialog yang muncul. Setelah dibuka, formulir PPDB akan aktif dan dapat diisi oleh calon peserta didik.

Bagaimana cara menutup PPDB? ▾

Saat PPDB sedang buka, tombol merah "Tutup PPDB 2027/2028" akan muncul. Klik tombol tersebut dan konfirmasi pada dialog. Setelah ditutup, Anda kembali dapat mengedit field formulir PPDB.

Apakah field formulir PPDB bisa diedit saat PPDB aktif? ▾

Tidak. Saat PPDB dalam status "Buka", pengeditan field formulir dinonaktifkan untuk mencegah perubahan yang dapat mengganggu proses pendaftaran yang sedang berjalan. Tutup dulu PPDB untuk dapat mengedit field.

Bagaimana menambah field baru di formulir PPDB? ▾

Saat PPDB dalam status tutup, klik Edit pada baris yang ingin diubah, atau gunakan tombol tambah di bawah tabel. Isi nama field, pilih tipe input (text, date, dropdown, dll.), dan atur apakah field wajib diisi atau tidak.

Apa arti tipe input pada field PPDB? ▾

Tipe input menentukan jenis data yang bisa dimasukkan pendaftar:

  • text – Teks bebas (nama, alamat)
  • date – Tanggal (tanggal lahir)
  • number – Angka (no. HP)
  • dropdown – Pilihan menu (jenis kelamin)
  • file – Upload dokumen/foto
  • email – Alamat email
⚙️ Mengelola Konten
Bagaimana cara mengedit profil sekolah? ▾

Masuk ke Manage → Profil Sekolah. Klik tombol "Edit" pada section yang ingin diubah. Ubah nama section, deskripsi, dan file yang ditautkan, lalu klik "Simpan".

Bagaimana cara upload foto ke Galeri? ▾

Buka Manage → Galeri. Klik "Edit Pict" pada foto yang ingin diganti, atau klik "+ Add Photo" untuk menambah foto baru. Pilih file gambar dari perangkat Anda, tambahkan keterangan (opsional), lalu klik "Simpan".

Bagaimana cara menandai event sebagai selesai di Kalender? ▾

Buka Manage → Kalender. Cari event yang ingin ditandai, lalu klik tombol "Mark Complete". Tombol akan berubah menjadi "✓ Done" dan event akan tampak lebih redup sebagai tanda telah selesai.

Apakah data yang dihapus bisa dipulihkan? ▾

Saat ini, penghapusan data bersifat permanen dan tidak dapat dipulihkan langsung dari dashboard. Pastikan Anda membuat backup data secara rutin melalui Settings → Lanjutan → Unduh Backup sebelum menghapus data penting.

Berapa banyak staff yang bisa ditambahkan? ▾

Tidak ada batasan jumlah staff yang dapat ditambahkan. Klik tombol "+ add more" di halaman Staff Pengajar & Tenaga Kependidikan untuk menambah data staff baru.

👤 Akun & Keamanan
Bagaimana cara mengganti password? ▾

Buka Manage Account atau Settings → Keamanan. Isi kolom "Password Lama" dengan password saat ini, lalu masukkan "Password Baru" dan konfirmasi ulang. Klik "Update Password" untuk menyimpan.

Bagaimana jika lupa password? ▾

Pada halaman login, klik link "Lupa Password". Masukkan email yang terdaftar dan ikuti instruksi yang dikirimkan ke email tersebut. Jika tidak berhasil, hubungi kepala sekolah atau pengelola sistem untuk reset password manual.

Bagaimana cara mencabut akses sesi yang mencurigakan? ▾

Buka Manage Account → bagian "Sesi Aktif". Anda dapat melihat semua sesi yang aktif beserta perangkat dan lokasinya. Klik tombol "Cabut" pada sesi yang tidak dikenal, atau gunakan "Cabut Semua Sesi Lain" untuk keamanan lebih.

Apa itu Autentikasi Dua Langkah (2FA)? ▾

2FA adalah lapisan keamanan tambahan. Selain password, Anda perlu memasukkan kode verifikasi yang dikirim ke HP atau aplikasi authenticator setiap kali login. Fitur ini dapat diaktifkan di Settings → Keamanan atau Manage Account.

🔧 Troubleshooting Teknis
Dashboard lambat atau tidak merespons? ▾

Coba langkah-langkah berikut:

  1. Refresh halaman (tekan F5 atau Ctrl+R)
  2. Bersihkan cache browser (Ctrl+Shift+Delete)
  3. Coba gunakan browser lain (Chrome, Firefox, Edge)
  4. Pastikan koneksi internet stabil
  5. Jika masalah berlanjut, hubungi tim teknis
Foto gagal diupload? ▾

Pastikan:

  • Format file yang didukung: JPG, PNG, WEBP
  • Ukuran file tidak melebihi 5MB
  • Koneksi internet stabil saat upload
  • Coba kompres foto terlebih dahulu menggunakan tools online
Tombol atau fitur tidak berfungsi? ▾

Kemungkinan penyebab:

  • Browser versi lama – perbarui ke versi terbaru
  • JavaScript dinonaktifkan – aktifkan di pengaturan browser
  • Ekstensi browser memblokir fungsi – coba mode Incognito
  • Cache yang rusak – bersihkan cache browser
Bagaimana melaporkan bug atau error? ▾

Gunakan formulir "Hubungi Kami" di halaman ini atau klik tombol "💬 Saran dan Masukan". Sertakan informasi:

  • Halaman dan fitur yang bermasalah
  • Langkah untuk mereproduksi error
  • Screenshot jika memungkinkan
  • Browser dan perangkat yang digunakan
